Maven

  • Archetype for DBFlute
  1. Generate 
    mvn archetype:generate \
    -DarchetypeCatalog=http://schatten4810.github.io/maven

    choose a jp.schatten.dbflute:dbflute-maven-archetype

    • groupId:{create maven groupId}
    • artifactId:{create maven artifactId}
    • version:{create maven version}
    • package:{project root pacakge}
    • clientProject:{dbflute client name. default is artifactId.}
    • container:{seasar,spring,guice,cdi}
    • containerVersion:{selected containers version.}
    • database:{h2,mysql,postgresql,oracle}
    • dbfluteVersion:{see DBFlute site.}
  2. Initialize DBFlute 
    cd <created artifactId>
    mvn -Pinit-dbflute
    or 
    mvn compile
